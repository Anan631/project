# اختبارات حساب شروش الأعمدة

## مثال تطبيقي شامل

### البيانات المدخلة:
- عدد الأعمدة: 12 عمود
- شكل العمود: مستطيل
- ارتفاع الشرش: 0.18 متر
- طول القاعدة: 1.20 متر
- عرض القاعدة: 1.20 متر
- عدد الطوابق: 4 طوابق
- مساحة البلاطة: 150 م²
- نوع المبنى: المباني السكنية (شقق ومنازل)

### الحسابات المتوقعة:

#### 1. حساب كمية الخرسانة
```
حجم شرش العمود الواحد = 1.20 × 1.20 × 0.18 = 0.2592 م³
إجمالي حجم الخرسانة = 0.2592 × 12 = 3.1104 م³
```

#### 2. حساب القيمة A
```
الحمل الميت = 7.0 كن/م²
الحمل الحي = 3.35 كن/م²
إجمالي الحمل = 7.0 + 3.35 = 10.35 كن/م²

A = (150 × 4 × 10.35) ÷ 0.195
A = 6210 ÷ 0.195
A = 31846.15
```

#### 3. تحديد أبعاد العمود المستطيل
```
العرض: B = √(31846.15 ÷ 2) = √15923.08 = 126.2 سم
الطول: C = 126.2 × 2 = 252.4 سم

النتيجة النهائية: 252.4 × 126.2 سم
```

## اختبارات الحالات الحدية

### اختبار 1: العمود الدائري مع قيمة A صغيرة
```
البيانات:
- A = 500
- شكل العمود: دائري

الحساب:
D = √(500 ÷ π) × 2 = √159.15 × 2 = 25.2 سم

النتيجة: القطر = 30 سم (الحد الأدنى)
```

### اختبار 2: العمود المربع مع قيمة A صغيرة
```
البيانات:
- A = 800
- شكل العمود: مربع

الحساب:
F = √(800 ÷ 2) = √400 = 20 سم

النتيجة: 35 × 35 سم (الحد الأدنى)
```

### اختبار 3: العمود المستطيل مع قيمة A صغيرة
```
البيانات:
- A = 300
- شكل العمود: مستطيل

الحساب:
B = √(300 ÷ 2) = √150 = 12.2 سم → 25 سم (الحد الأدنى)
C = 25 × 2 = 50 سم

النتيجة: 50 × 25 سم
```

## اختبارات التحقق من صحة المدخلات

### اختبار ارتفاع الشرش
- ✅ 0.15 متر: مقبول
- ✅ 0.18 متر: مقبول
- ✅ 0.20 متر: مقبول
- ❌ 0.14 متر: مرفوض (أقل من الحد الأدنى)
- ❌ 0.21 متر: مرفوض (أكبر من الحد الأقصى)

### اختبار القيم الرقمية
- ❌ عدد الأعمدة = 0: مرفوض
- ❌ مساحة البلاطة = -10: مرفوض
- ❌ عدد الطوابق = 0: مرفوض

## نتائج متوقعة لأنواع مباني مختلفة

### مبنى تجاري (محلات وأسواق)
```
البيانات: مساحة البلاطة = 200 م², عدد الطوابق = 3
الأحمال: ميتة = 7.0, حية = 6.0, إجمالي = 13.0 كن/م²
A = (200 × 3 × 13.0) ÷ 0.195 = 40000

عمود مستطيل:
B = √(40000 ÷ 2) = 141.4 سم
C = 141.4 × 2 = 282.8 سم
النتيجة: 282.8 × 141.4 سم
```

### مستشفى
```
البيانات: مساحة البلاطة = 300 م², عدد الطوابق = 5
الأحمال: ميتة = 7.5, حية = 5.4, إجمالي = 12.9 كن/م²
A = (300 × 5 × 12.9) ÷ 0.195 = 99230.77

عمود دائري:
D = √(99230.77 ÷ π) × 2 = 355.4 سم
النتيجة: 355.4 سم (قطر)
```
