# دليل حساب شروش الأعمدة

## نظرة عامة
هذا النظام يقوم بحساب كميات الخرسانة في شروش الأعمدة وتحديد أبعاد الأعمدة وفقاً للمعايير الهندسية المعتمدة.

## خطوات الحساب

### 1. حساب كمية الخرسانة في شروش الأعمدة

#### أ) حساب كمية الخرسانة في شرش العمود الواحد
```
حجم خرسانة شرش العمود الواحد = طول القاعدة × عرض القاعدة × ارتفاع الشرش
```

#### ب) حساب كمية الخرسانة في جميع شروش الأعمدة
```
إجمالي حجم خرسانة شروش الأعمدة = حجم خرسانة شرش العمود الواحد × عدد الأعمدة
```

### 2. حساب القيمة A لتحديد أبعاد العمود
```
A = (مساحة البلاطة × عدد الطوابق × الأحمال) ÷ 0.195
```

**ملاحظة:** قيمة الأحمال (الميتة + الحية) يتم جلبها تلقائياً من قاعدة البيانات بناءً على نوع المبنى.

### 3. تحديد أبعاد العمود حسب الشكل المختار

#### أ) العمود المستطيل
```
العرض: B = √(A ÷ 2)
- إذا كانت قيمة B ≥ 25 سم → العرض = B
- إذا كانت قيمة B < 25 سم → العرض = 25 سم

الطول: C = B × 2
- إذا كانت قيمة C ≥ 50 سم → الطول = C
- إذا كانت قيمة C < 50 سم → الطول = 50 سم

النتيجة: الطول × العرض (سم)
```

#### ب) العمود الدائري
```
القطر: D = √(A ÷ π) × 2
- إذا كانت قيمة D ≥ 30 سم → القطر = D
- إذا كانت قيمة D < 30 سم → القطر = 30 سم

النتيجة: القطر (سم)
```

#### ج) العمود المربع
```
البعد: F = √(A ÷ 2)
- إذا كانت قيمة F ≥ 35 سم → العرض = F
- إذا كانت قيمة F < 35 سم → العرض = 35 سم

الطول = العرض

النتيجة: الطول × العرض (سم)
```

### 4. تخزين النتائج
- يتم تخزين أبعاد العمود النهائية (حسب الشكل المختار) في قاعدة البيانات
- تكون القيم المخزنة جاهزة للاستخدام في التقارير أو الصفحات الأخرى

## قيود النظام

### ارتفاع الشرش
- يجب أن يكون بين 15-20 سم (0.15 - 0.20 متر)

### الحد الأدنى لأبعاد الأعمدة
- العمود المستطيل: عرض ≥ 25 سم، طول ≥ 50 سم
- العمود الدائري: قطر ≥ 30 سم
- العمود المربع: بعد ≥ 35 سم

## مخرجات النظام

1. **إجمالي حجم الخرسانة** (متر مكعب)
2. **أبعاد العمود النهائية** (حسب الشكل المختار)
3. **القيمة A المحسوبة**
4. **تفاصيل الأحمال المستخدمة**
5. **تقرير شامل قابل للحفظ والطباعة**

## استخدام النظام

1. أدخل عدد الأعمدة
2. اختر شكل العمود (مستطيل/دائري/مربع)
3. أدخل ارتفاع الشرش (0.15-0.20 متر)
4. أدخل أبعاد القاعدة (الطول والعرض)
5. أدخل عدد الطوابق ومساحة البلاطة
6. اختر نوع المبنى
7. اضغط "حساب كميات الشروش"
8. راجع النتائج واحفظها في التقارير
